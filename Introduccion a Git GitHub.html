Introduccion Git y utilizacion en Github

¿que es? ¿que no es?
* Git es un sistema que administra o controla las distintas versiones de un programa.a lo largo de la evolucion del programa * Un administrador netamente con funcionalidades de tipo publico
* es una aplicacion openSourse, desarrollado poro linus torvalds para controlar las contribuciones en el kernel de linux * para trabajar y desbloquear funcionalidades se debe elegir un plan de pago y categoria
* detalla un listado de cambios que se hallan realizado en el programa * cuando es publico el repositorio puede ser descargado y modificado sin ninguna restricción
es un sistema de control distribuido: permite la cordinacion con otros usuarios para la alteracion del codigo por medio de copias locales de cada participante, concervacion de cambios y posterior adicion al programa principal en el repositorio remoto o local pueden existir fallo importantes cuando se renombran los commits porque no se sabe cual es el commit principal
